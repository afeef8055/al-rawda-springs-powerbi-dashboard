# Al-Rawda-Springs-Powerbi-Dashboard

Al Rawda Springs is a bottled water company focused on efficient production, quality control, and delivering high-quality drinking water across multiple regions.


📊 Power BI Dashboard Project | Al Rawda Springs Analysis
 Interactive Power BI dashboard built using real-world data from Al Rawda Springs 💧

🔹 What I worked on:
🧹 Data cleaning & transformation using Power Query
🔗 Data modeling for structured analysis
📈 Interactive data visualization for better decision-making

🔹 Dashboard Highlights:
🏭 Product Analysis – production trends, defect rates, and energy usage
💰 Sales Analysis – revenue, profit margin, and regional performance
⚙️ Operations Performance – downtime, cost analysis, and energy consumption
🎛️ Dynamic filters for month and product type for deeper insights

🔹 Key Insights:
📊 Identified high-performing products and regions
📉 Analyzed defect rates across shifts and product types
⚡ Evaluated operational efficiency and cost drivers

This project helped me strengthen my skills in Power BI, data cleaning, DAX, and business storytelling 🚀
